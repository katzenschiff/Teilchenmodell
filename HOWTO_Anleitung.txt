// Teilchenmodell
// particle model
//
// demonstrate a simple particle model for students where 
// all matter is made of particles
// all particles are in motion the whole time
// increased temperature equals increased movement of the particles
// This is my first JS script so bear with me 
// 
// 
// open in browser (Chrome, Firefox)
// up arrow: warmer + more red
// down arrow: cooler + more blue
// space bar: toggle pause
//

Visualisiert die Bewegung der Teilchen im Teilchenmodell
Anleitung:

im Browser öffnen (Chrome, Firefox)

Stop/start: Leertaste
heisser: Pfeil nach oben
kälter: Pfeil nach unten

geplant:
Thermometer visualisieren
Aggregatszustände zufügen
mehr Knöpfe

bugs:
nicht zu oft auf Pfeil drücken, ich fange die color codes nicht ab. :D
